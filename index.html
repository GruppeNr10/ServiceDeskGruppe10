Hvordan lage en nettside med Github

Vil du lage din egen nettside, men finner ikke ut hvordan. Da er dette nettsida for deg.
Her kan du finne ut hvordan du går fra litt simpel kode til din egen nettside lagt ut via github!

Steg 1: Hvordan komme igang?

Det første du må gjøre er å gå til GitHub sin hjemeside.
Lag en bruker, Og fortsett til neste steg

Steg 2: Starten på et prosjekt

Etter du har lagd brukeren din vil du få se denne sida. Trykk på Start a Project for å gå til neste steg.

Skriv in navnet til nettsida di. Du kan alltid forandre dette senere. Du kan legge til en beskrivelse.
Sjekk at boksen Initialize this repository with a README er krysset av. Så trykker du på Create repository.

Steg 3: Branches

Trykk på Branch. Og lag en ny branch med navn Ph-pages.

Gå til Settings -> Branches. Setter du Default branch til ph-pages. Gå så tilbake til Code.

Steg 4: Koden

Trykk på Create new file. Gi filen navnet index.html. Her kan du kopiere inn koden du vil nettsida di skal ha.
Etter dette trykker du på Commit new file nederst på siden.

Tilslutt går du inn på settings og blar ned til du finner GitHub Pages. Kopier linken og lim den inn i adresse boksen. Du kan nå se din egen siden!
